GenXLink Portable v1.0.0
============================

This is a portable version of GenXLink Remote Desktop Platform.
No installation required - just run GenXLink-Portable.bat!

Features:
- Standalone API Server
- WebSocket Signaling Server
- WebRTC Media Support
- Zero Configuration

System Requirements:
- Windows 10/11 (x64)
- 100MB RAM
- 50MB Disk Space
- Network Connection

Usage:
1. Double-click GenXLink-Portable.bat
2. Wait for servers to start
3. Open browser to shown addresses
4. Start remote desktop sessions
