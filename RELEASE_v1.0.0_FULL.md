#  GenXLink v1.0.0 - First Stable Release

##  Created in India  Crafted by Indians 

GenXLink is a modern, cross-platform remote desktop application built with Rust, featuring end-to-end encryption and WebRTC peer-to-peer streaming.

##  Download Options

###  Portable Version (Recommended)
`GenXLink-v1.0.0-Windows-Portable.zip`

Perfect for:
- Quick testing and evaluation
- USB drives and portable use
- No installation required

**Instructions:**
1. Download and extract ZIP
2. Double-click `GenXLink.exe`
3. Ready to use!

###  Windows Installer (NSIS)
`GenXLink-v1.0.0-Setup.exe`

For:
- Standard Windows installation
- Start menu shortcuts
- Desktop icon

###  MSI Package (Enterprise)
`GenXLink-v1.0.0.msi`

For:
- Group Policy deployment
- SCCM/Intune
- Silent install: `msiexec /i GenXLink-v1.0.0.msi /qn`
