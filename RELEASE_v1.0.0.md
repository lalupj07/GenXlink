## GenXLink v1.0.0 - First Stable Release

### Features
- **Full UI Application** with dashboard, connections, sessions, file transfer
- **8 Languages**: English, Spanish, French, German, Chinese, Japanese, Hindi, Malayalam
- **Themes**: Dark, Light, System
- **WebRTC P2P** connections with TURN/STUN for NAT traversal
- **Real File Transfer** with progress tracking
- **Live Signaling Server** on Railway

### Downloads
| File | Description |
|------|-------------|
| GenXLink-v1.0.0-Windows-Portable.zip | Portable - No install needed |
| GenXLink-v1.0.0-Setup.exe | Windows Installer (NSIS) |

### Requirements
- Windows 10/11 (64-bit)

### Quick Start
1. Download the portable ZIP or installer
2. Run GenXLink.exe
3. Connect to devices on your network
